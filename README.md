# reversible-hyperinf
Reversible evolutionary approaches with EvAM

If observations are generated by a reversible evolutionary process, how much information do we lose if we model it as irreversible? 

Uses hyperinf https://github.com/StochasticBiology/hyperinf and several other libraries.

`rev-challenge.R` looks at small-scale case studies where reversibility challenges aspects of the EvAM inference. `rev-test-large-hhmm.R` is a larger simulation study involving larger, more diverse generative mechanisms.
